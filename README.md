Tienda “Las recetas de la abuela”

Proyecto web simple de tienda en línea que funciona 100% en el navegador usando HTML, CSS, JavaScript y LocalStorage. Permite mostrar productos, gestionar un carrito y simular un pago, con precios en COP.

Características:

1. Catálogo dinámico de productos.

2. Carrito persistente con LocalStorage.

3. Resumen del pedido y pago simulado.

4. Precios formateados en Pesos Colombianos.

5. Posibilidad de agregar productos (si se incluye un panel de administración).

Estructura
index.html       # Catálogo
carrito.html     # Carrito de compras
pago.html        # Pago simulado
assets/          # Imágenes
js/              # app.js, cart.js, pago.js
css/             # style.css

Instalación / Uso

1. Clonar el repositorio

git clone <repo-url>


2. Ejecutar

Puedes abrir index.html directamente

Restablecer datos

Requisitos

Ninguno. Solo un navegador moderno.
